Urban Garden Planner

A mobile-friendly tool that helps small-space gardeners plan crop rotations and watering schedules.

Description

Urban Garden Planner is designed for balcony and backyard gardeners. It provides smart suggestions for plant placement based on sunlight, season, and companion planting.

The app is built with a component-based architecture and supports offline mode. The goal is to simplify weekly care with reminders and checklists.
